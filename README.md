## 1. Проверка email-доменов (`1. Проверка email-доменов`)
- Скрипт `check_mail.py` проверяет MX-записи домена для списка email-адресов.
- Список email можно поменять прямо в файле.

## 2. Мини-интеграция с Telegram (`2. Мини-интеграция с Telegram`)
- В папке — workflow `n8n_workflow.json` и скриншот.
- Что делает: триггер Google Drive реагирует на новый файл в папке, качает его, извлекает текст и шлёт содержимое в Telegram.
- Импортируйте workflow в n8n, подставьте свои креды Google Drive и Telegram chat ID.

## 3. Архитектурная задача (`3. Архитектурная задача`)
- `architecture.md` — описание простой инфраструктуры для 1200 email-адресов
